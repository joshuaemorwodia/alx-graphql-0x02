# ALX Rick and Morty App

This project is a Next.js application that uses Apollo Client to consume the Rick and Morty GraphQL API.

## Setup

- TypeScript
- ESLint
- Tailwind CSS
- Apollo Client

## Structure

- `graphql/apolloClient.ts`: Apollo Client setup
- `graphql/queries.ts`: GraphQL query for fetching episodes
- `pages/_app.tsx`: App component wrapped in ApolloProvider
